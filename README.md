# Simple AI Chatbot (WordPress Plugin)

A lightweight WordPress plugin that integrates a local AI chatbot using **Ollama (LLM)** and provides a real-time chat UI via shortcode.

---

## 🚀 Features

- Local AI chatbot using Ollama (`llama3` model)
- WordPress shortcode integration
- Real-time chat interface (AJAX-based)
- Chat history saved in browser (localStorage)
- Enter key support for quick messaging
- Simple, responsive UI
- No external API costs

---

## 📦 Requirements

Before using this plugin, make sure you have:

- WordPress installed
- PHP 7.4+
- Ollama installed locally  
  👉 https://ollama.com
- A running model (example: `llama3`)

Run model:
```bash
ollama run llama3
